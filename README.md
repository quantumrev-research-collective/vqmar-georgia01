## Planned contents

This repository will contain the reference implementation for the VQ-MAR Georgia pilot, including:

- **Data acquisition pipeline** for five public geospatial sources: USDA SSURGO (soil), USGS NWIS (groundwater),
  USGS NLCD (land cover), NOAA NCEI (climate), and OpenStreetMap (infrastructure)
- **QUBO assembly** for managed aquifer recharge site selection
- **Solver implementations**: greedy, simulated annealing, and genetic algorithm classical baselines
- **QAOA pathway** with warm-start initialization, executed on Qiskit Aer simulators and validated on the
  IBM Kingston superconducting QPU at N = 20 and N = 50
- **MPS simulation** at N = 50 with parameter-concentration analysis
- **Public-data documentation** for each source
